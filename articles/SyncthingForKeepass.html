Syncthing for keeping files synced accross various devices

Syncing services like Dropbox or Google Drive add any limitations they like to the services they offer. Those limitations often have to do with drive size limits, the software they build for different platforms and finally not to mention hold you to their Terms and Conditions.

I was looking for a way to keep my Keepass database synced accross different devices. For now the best solution and one I have now tested for a only a few hours is syncthing. The project is open source and can be followed on github in this link.

There are a few things about syncthing which are great:

  1. They have support for all platforms (it seems that ios is still missing but it's being worked on.
  2. It's P2P which means that my keepass db doesn't have to sit on a server.
  3. It's super light and easy to use. The people building it have kept the interface a small piece of HTML which makes it super light and easy to update.
  4. The project has been going on and stable for over 4 years.
